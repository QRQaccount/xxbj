二、中断系统

    1、概念：
        引起中断的事件称为中断源。中断源向CPU提出处理的请求称为中断请求。
        发生中断时被打断程序的暂停点成为断点。CPU暂停现行程序而转为响应中断请求的过程称为中断响应。
        处理中断源的程序称为中断处理程序。CPU执行有关的中断处理程序称为中断处理。而返回断点的过程称为中断返回。
    
        52单片机共有6个中断源
        1、INT0 外部中断0，由P3.2口引入，低电平或下降沿引起      中断级别0
        2、INT1 外部中断1 由P3.3端口线引入，低电平或下降沿引起   中断级别2
        3、T0 定时器计数器0中断，由T0计数器记满回零引起          中断级别1
        4、T1 定时器/计数器1中断，由T1计数器记满回零引起         中断级别3
        5、T2定时器/计数器2中断，由T2计数器记满回零引起          中断级别5
        6、TI/RI串行口中断，串行口接受到一帧字符时引起中断       中断级别4
    
    
    2、设置中断有关的寄存器
        什么是特殊功能寄存器：
            51单片机内部有一个CPU用来运算、控制，有四个并行I/O口，分别是P0、P1、P2、P3，有ROM，
            用来存放程序，有RAM，用来存放中间结果，此外还有定时/计数器，串行I/O口，中断系统，
            以及一个内部的时钟电路。在单片机中有一些独立的存储单元是用来控制这些器件的，
            被称之为特殊功能寄存器(SFR)
    
        1、中断允许器IE
            用于设定各个中断源的打开与关闭，IE在特殊功能寄存器中，字节地址为A8H，位地址
            为A8H到AFH。
            位符号     位地址     功能
            EA         AFH     EA = 1时全局中断控制打开，再由各自的中断控制位控制
                                中断的打开和关闭
            ET2        ADH      ET2 = 1，T2中断打开；ET2 = 0，T2中断关闭
            ES         ACH      ES = 1，串行口中断打开，ES = 0，串行口中断关闭
            ET1        ABH      ET1=1，T1中断口打开；ET=0，T1中断口关闭
            EX1        AAH      EX1=1，外部中断1打开；EX1=0，外部中断1关闭
            ET0        A9H      ET0=1，T0中断打开；ET0=0，T1中断关闭
            EX0        A8H      EX0=1，外部中断0打开；EX0=0，外部中断0关闭
        2、中断优先级寄存器IP
            用于设定中断源属于两级中断中的哪一级，位于特殊寄存器。字节地址B8H，位地址
            B8H到BFH
    3、单片机的定时器中断
        THX和TLX
        定时寄存器的实质是加一计数器，由高八位和低八位两个寄存器组成，THX和TLX(X代表1或0)
        特殊寄存器：
            TMOD用于控制寄存器/定时器的工作方式
            TCON控制T0，T1的启动停止和设置溢出标志
        计数器的输入的数据来源有两个，一个是系统时钟震荡，还有一个是T0/T1口的信号输入，每接受一个脉冲计数器就加一
        当计数器全部加为1的时候再进行一个数据输入则使计数器归零，且计数器溢出会使TCON寄存器的TF0或TF1置1，向cpu发出中断请求
    
        溢出时计数器的值减去计数的初值才是计数值
        初值计算方法：
            向THX中输入(65536-N)/256
            向TLX中输入(65536-N)%256
        1、定时模式
            加一计数器是对内部机器周期计数，计数值乘以周期就是定时时间
        2、计数器模式
            外部计数器的脉冲由T0或T1引脚输入计数器。在机器周期的S5P2期间采样T0，T1。当某周期采样到T0,T1高电平输入
            而在下一个周期采样到低电平，则计数器加一。更新的数值在S3P1期间装入计数器
            因为检测下降沿需要两个周期，所以被采样的电平至少要维持一个机器周期。即当晶振频率为12Mhz的时候，最高的计数频率1/2MHz
        3、计数器/定时器的工作方式的设定
            通过设置两个特殊寄存器：TCON，TMOD
            1、TMOD的设置：
            字节地址为89H不能位寻址
                TMOD寄存器中D7~D4用于控制定时器1，D3~D0用于控制定时器0
                从高到低分别对应
                    GATE C/T M1 M0
                    GATE:
                        门控制位，当GATE=1的时候定时器/计数器的启动由TCON的TR0（或TR1）和外部的中断引脚上的电平来共同控制
                        GATE=0，定时器/计数器的启动与停止只受TR0（或TR1）来控制
                    C/T：
                        C/t=1，为计数器模式
                        C/T=0，为定时器模式
                    M1、M0：
                        工作方式选择位
                        M1  M0  工作方式
                        0   0       0       13位定时/计数器
                        0   1       1       十六位定时/计数器
                        1   0       2       8位初值自动重装的8位定时/计数器
                        1   1       3       仅适用于T0，分成两个8位计数器，T1停止工作
            2、TCON的设置：
                字节地址为88h，位地址88H~8FH，可进行位寻址
                D7~D0分别对应
                TF1 TR1 TF0 TR0 IE1 IT1 IE0 IT0
                在此以定时器1位例
                TF1：
                    溢出标志位，当定时器记满溢出的时候，由硬件使TF1置1，在进入中断程序后，由硬件置0
                    在定时器模式下，不用人为的操作。若是用软件查询的方式置1，则需要用软件将此置0
                TR1：
                    运行控制位，由软件清零关闭定时器。当GATE=1且INT1为高电平的时候，TR1置1，启动定时器；当GATE=0，TR置1启动定时器
                IE1:
                    外部中断请求标志，
                    当IT1=0，每个机器周期S5P2采样INT1引脚，若INT1引脚为低电平则置1，否则，清零
                    当IT1=1，负跳变时IE1置1
                    当IE1=1的时候，向cpu申请中断
                IT1：
                    参考IE1的说明
    4、实际的程序编写流程
        使用c语言进行编写的时候，使用interrupt 中断号 using 工作组
        1、外部中断
            声明相应的中断允许打开
            以外部中断1为例
            EA=1;
            EX0=1;
            TR0 = 0;
        2、定时器中断
            1、对TMOD进行赋值，以确定T0或T1的工作方式（使用TMOD=[16位数]）
            2、计算初值，并将初值写入THX，TLX中
            3、对IE进行赋值（EA，ETX），开放中断
            4、使用TR0或TR1进行定时或计数

机器周期
1、每个状态周期有P1和P2两个相,每个相占一个振荡周期,故一个机器周期为12个振荡周期.
2、在每个时钟周期（即机器状态时间S）的前半周期，相位1（即P1信号）有效，在每个时钟周期的后半周期，相位2（即P2信号）有效。
3、每个时钟周期（S）有两个节拍即相位P1和P2(可理解为P1：高电平，P2：低电平)，CPU以两相时钟P1和P2为基本节拍指挥8051各们部件协调工作。

时间周期

2. 11.0592MHz
